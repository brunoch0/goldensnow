| 지표 | 값 | 의미 |
|------|-----|------|
| Dubai Metro 일일 이용객 | **750K+** | RTA 메트로를 매일 지나는 노출 인구 |
| Dubai Mall 연간 방문객 | **100M+** | 세계 최대 쇼핑몰, 고소비층 집결 |
| 두바이 거주 인구 | **3.7M+** | 전 세계 자본·인재가 모이는 도시 |
| 중동 프리미엄 OOH 시장 | **#1** | 중동 옥외광고의 정점 |

> ※ 수치는 goldensnow-ooh 라이브 사이트 기준이며, 실제 제안 시 최신 데이터로 업데이트됩니다.

**왜 지금인가**
- 두바이는 한국 브랜드에 대한 관심과 구매력이 동시에 높은 시장
- 디지털 빌보드 보급으로 짧은 기간·유연한 집행 가능
- 랜드마크 노출 = 브랜드 위상 상승 효과

## 4. 집행 프로세스

```
01. 목표·예산 상담  →  02. 미디어 선정·예약  →  03. 소재 제작  →  04. 집행·리포트
```

1. **목표·예산 상담** — 타겟·기간·예산 기반 미디어 믹스 제안
2. **미디어 선정·예약** — RTA·몰·Emaar 최적 위치 확보
3. **소재 제작** — 두바이 현지 규정에 맞춘 광고 소재 제작
4. **집행·리포트** — 캠페인 집행 후 노출·성과 리포트 전달
