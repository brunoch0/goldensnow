## 1. 프로젝트 개요

**Dubai 37**은 누구나 **정확히 37초** 길이의 영상으로 참여하는 글로벌 초단편 영화 챌린지이자, 두바이라는 도시를 다양한 시선으로 기록하는 **참여형 도시 문화 아카이브**입니다. 일회성 이벤트가 아니라 시즌제로 계속 축적되는 프로젝트입니다.

**목표**
- 37초 필름으로 두바이의 도시 에너지와 미래 비전을 재해석
- 글로벌 크리에이터 참여로 문화적 engagement 확대
- 반복 가능한 시즌 구조로 장기 영상 아카이브 구축
- 도시 브랜딩과 크리에이티브 생태계 연결

> "경쟁 중심의 상업 페스티벌이 아니라, 참여형 도시 문화 아카이브 플랫폼."

## 2. 컨셉 및 Why 37

**Season 1 컨셉 — "두바이에 존재했던 순간"**
잘 만든 이야기가 아니라, 지금 이 도시 안에서의 37초를 남기는 것. 광고가 아니고, 설명이 아니며, **도시의 한 장면**이다.

**Why 37 (dubai37.com 기준)**
- **인간의 체온, 37°C** — AI 시대에 가장 인간적인 시간의 단위에서 이야기를 시작한다
- **3 = 구조** (과거/현재/미래 · 시간/공간/물질 · 시작/중간/끝)
- **7 = 완성** (일주일 7일 · 무지개 7색 · 음계 7음)
- **37 = 구조(3) + 완성(7)** → 짧지만 완결된 이야기

> 보조 키메시지(초기안): "37 Seconds. One City. Infinite Perspectives."

## 5. 참여 방법

**참여 흐름**
1. 37초 숏폼 필름 제작
2. 개인 유튜브 채널에 업로드
3. dubai37.com 에서 지원서 제출
4. 1차 검토 및 승인
5. 홈페이지 아카이브 게시
6. 공개 심사 및 최종 선정

**출품 규격**
| 항목 | 기준 |
|------|------|
| 러닝타임 | **정확히 37초** (±0.5초 허용) |
| 해상도 | FHD 1920×1080 |
| 프레임레이트 | 24fps 또는 30fps |
| 파일 형식 | MP4, MOV (H.264 권장) |
| 화면 비율 | 가로·세로 무관 |
| 자막 | 영어 자막 필수 (대사 있을 시) |
| 언어 | 제한 없음 |
| 엔딩 크레딧 | **DXB37 표기 필수** |
| 장비 | 제한 없음 (스마트폰/카메라/AI 허용) |

**지원서 항목:** 스태프 실명 · 작품 제목 · 촬영 장비 · 촬영 장소 · 유튜브 링크 · 저작권 동의

**저작권 / AI 정책**
- 모든 작품의 저작권은 제작자에게 귀속, Dubai 37은 아카이빙·홍보 사용권만 보유 (2차 활용 시 크레딧 필수)
- Season 1은 AI 활용 제한 없음 (단, 저작권 침해 요소 포함 시 제외)

**심사 요소:** 관객 반응(조회수·좋아요) · 컨셉 명확성 · 37초 활용도 · 감정적 존재감
