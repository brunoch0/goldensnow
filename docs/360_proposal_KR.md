## 1. 서비스 개요

360° Digital Twin은 전문 장비 **Matterport Pro3** 로 실제 공간을 밀리미터 단위로 스캔하여, 누구나 웹·모바일에서 실제로 걸어다니듯 둘러볼 수 있는 **인터랙티브 3D 디지털 자산**으로 만드는 서비스입니다.

단순 360° 파노라마 사진이 아니라:

- **실측 기반 3D 모델** — 도면·치수 자동 추출
- **정보 태그(Tag)** — 공간 안에 제품·가격·설명·링크 삽입
- **임베드** — 홈페이지·랜딩페이지·메신저에 링크 하나로 삽입
- **VR 지원** — 헤드셋으로 몰입형 경험 제공

> 한 번 제작하면 24시간, 전 세계 어디서나 작동하는 영업 사원이 됩니다.

## 3. 패키지 (상담 후 확정)

정확한 견적은 **공간 규모·촬영 범위·후작업 수준**에 따라 달라집니다.
아래는 구성 방향이며, 상담을 통해 맞춤 견적을 제공합니다.

| 구분 | 대상 | 포함 |
|------|------|------|
| **Basic** | 단일 소규모 공간 | 스캔 + 기본 디지털 트윈 + 공유 링크 |
| **Standard** | 중규모 / 다실 공간 | 스캔 + 정보 태그 + 도면 + 웹 임베드 |
| **Premium** | 대규모 / 브랜딩 필요 | 풀 스캔 + 브랜딩 + 동선 설계 + 운영/분석 지원 |

> 💡 단가는 상담 후 확정 방식입니다. 공간 정보를 알려주시면 정확한 견적을 드립니다.

## 4. 진행 프로세스

```
01. 상담·현장 분석   →   02. Matterport Pro3 촬영
        ↓                          ↓
04. 납품·운영 지원   ←   03. 디지털 트윈 제작(태그·도면·브랜딩)
```

1. **상담·현장 분석** — 공간 규모와 목적 파악, 촬영 범위 설계
2. **Matterport Pro3 촬영** — 실측 기반 고정밀 3D 스캔
3. **디지털 트윈 제작** — 태그·동선·브랜딩 적용, 인터랙티브 자산 완성
4. **납품·운영 지원** — 웹 임베드, 공유 링크, 분석 연동

표준 소요 기간: 공간 규모에 따라 **약 1~3주**
